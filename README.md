# Dashboard Indikator Kemiskinan Indonesia (2021–2025)

Visualisasi interaktif data kemiskinan 38 provinsi Indonesia menggunakan Power BI, berdasarkan data resmi Badan Pusat Statistik (BPS).

---

## Tools yang Digunakan

| Tool | Fungsi |
|---|---|
| Microsoft Excel | Pengolahan dan pembersihan data mentah dari BPS |
| Power BI Desktop | Pembuatan visualisasi interaktif dan dashboard |

---

## Sumber Data

- **Sumber:** Badan Pusat Statistik (BPS) — [www.bps.go.id](https://www.bps.go.id)
- **Periode:** 2021 – 2025
- **Cakupan:** 38 Provinsi Indonesia
- **Unit analisis:** Provinsi (tahunan)

### Variabel yang digunakan

| Variabel | Keterangan |
|---|---|
| % Kemiskinan | Persentase penduduk miskin per provinsi |
| Jumlah Penduduk Miskin | Jumlah penduduk miskin (dalam ribu jiwa) |
| IPM | Indeks Pembangunan Manusia (%) |
| TPT | Tingkat Pengangguran Terbuka (%) |

---

## Fitur Dashboard

- **Filter tahun** — pilih tahun analisis (2021–2025) untuk memperbarui semua visual secara dinamis
- **Filter provinsi** — lihat tren tahunan untuk satu atau beberapa provinsi sekaligus
- **Ringkasan KPI** — rata-rata penduduk miskin, TPT, IPM, dan persentase kemiskinan nasional
- **Tren persentase kemiskinan** — grafik garis perkembangan 2021–2025 per provinsi
- **Bar chart penduduk miskin** — peringkat provinsi dengan jumlah penduduk miskin tertinggi
- **Scatter plot IPM vs Kemiskinan** — hubungan antara IPM dan tingkat kemiskinan per provinsi

---

## Cara Penggunaan

1. **Buka file Power BI** — unduh dan buka file `.pbix` menggunakan Power BI Desktop.
2. **Refresh data (opsional)** — jika ingin memperbarui data, pastikan file Data sumber berada di path yang sama, lalu klik *Refresh*.
3. **Gunakan filter interaktif** — pilih tahun di dropdown kiri atas, atau pilih provinsi di panel tengah untuk menyaring tampilan.
4. **Eksplorasi visual** — klik titik di scatter plot atau bar di chart untuk highlight silang antar visual.
